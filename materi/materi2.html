APA ITU BASIS DATA?

Menurut saya basis data itu mirip seperti lemari penyimpanan informasi. Kalau lemari untuk menyimpan baju atau buku, basis data menyimpan data digital.

Contohnya:

  • Di kampus ada daftar nilai semua mahasiswa, disimpan di basis data.
  • Aplikasi belanja online menyimpan data barang, harga, dan riwayat belanja.

Tujuan basis data adalah supaya data rapi, mudah dicari, dan aman. Kita tidak perlu repot mencari manual. Tinggal pakai perintah SQL untuk menambah, mencari, mengubah, atau menghapus data.
